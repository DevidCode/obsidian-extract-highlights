# Extract Highlights to Note

Crée une note épurée contenant uniquement les passages surlignés (`==...==`) de la note active.
Pensé pour distiller une transcription vidéo (YouTube transcript fetcher) en ne gardant que les highlights, avant déconstruction.

- Commande : **Extraire les surlignages → nouvelle note** (+ icône surligneur dans le ruban)
- Réglages : garder les timestamps · supprimer la note originale · dossier de destination
- Compatible **mobile, PC et bureau distant** (`isDesktopOnly: false`)

## Installation via BRAT
Ajouter ce dépôt dans **Obsidian42 - BRAT** → *Add beta plugin*.
